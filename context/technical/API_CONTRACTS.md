# API Contracts

> Generated by setup wizard from PRD. Expand as endpoints are built.

## Conventions

- **Base URL:** `http://localhost:8000/api` (dev), configurable via env
- **Auth:** None in v1 (public API)
- **Error format:** `{ "error": "message", "detail": "optional detail" }`
- **Content-Type:** `application/json` for REST, `text/event-stream` for SSE

## Endpoints

### Episodes

#### `GET /api/episodes`
List all episodes, newest first.

**Response:** `Episode[]` (without full script, just metadata)

#### `GET /api/episodes/{date}`
Get a single episode by date (e.g. `2026-03-24`).

**Response:** Full `Episode` object including products and script.

#### `GET /api/episodes/{date}/audio`
Stream the episode MP3 file.

**Response:** `audio/mpeg` binary stream

### Generation

#### `POST /api/generate`
Trigger episode generation for today.

**Request body (optional):**
```json
{
  "preferences": {
    "interests": ["Open Source", "LLMs"],
    "deep_dives_only": false,
    "technical_detail": true
  }
}
```

**Response:** `{ "episode_id": "2026-03-24", "status": "generating" }`

#### `GET /api/generate/progress`
SSE endpoint for real-time generation progress.

**Event types:**
- `scraping_ph` — Started scraping Product Hunt
- `product_found` — A product was scraped (includes product data)
- `scraping_website` — Currently scraping a product website (includes URL)
- `website_scraped` — Product website scrape complete (includes enriched product data)
- `generating_script` — LLM is generating the podcast script
- `script_ready` — Script generation complete
- `synthesizing_voice` — TTS in progress (includes speaker name and line index)
- `stitching_audio` — FFmpeg stitching audio clips
- `complete` — Episode ready (includes episode metadata)
- `error` — Something went wrong (includes error message)

### Products (within episodes)

#### `GET /api/episodes/{date}/products/{name}`
Get deep-dive data for a specific product from an episode.

**Response:** Full `Product` object with website_content and all extracted data.
