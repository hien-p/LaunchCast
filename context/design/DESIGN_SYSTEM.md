# Design System

> Generated by setup wizard from PRD.

## Direction & Tone

**Feel:** Casual, opinionated, fun — like two friends at a coffee shop talking tech.
**Aesthetic:** Modern, dark-mode-first, with bold accent colors. Think podcast app meets tech blog.
**Not:** Corporate, sterile, enterprise. Never feel like a dashboard or admin panel.

Keywords: energetic, opinionated, clean, techy, approachable.

## Colour Palette

### Primary
- **Background:** `#0A0A0B` (near-black)
- **Surface:** `#141416` (card backgrounds)
- **Surface Elevated:** `#1C1C20` (modals, sticky player)

### Accent
- **Aero (Skeptic):** `#F97316` (warm orange — analytical, direct)
- **Nova (Optimist):** `#8B5CF6` (vibrant purple — creative, forward-looking)
- **Primary Action:** `#3B82F6` (blue — CTAs, links)

### Semantic
- **Success:** `#22C55E`
- **Warning:** `#EAB308`
- **Error:** `#EF4444`

### Text
- **Primary:** `#F5F5F5`
- **Secondary:** `#A1A1AA`
- **Muted:** `#71717A`

## Typography

- **Headings:** Inter (bold, tight tracking)
- **Body:** Inter (regular, 16px base)
- **Monospace:** JetBrains Mono (code snippets, ratings, stats)
- **Scale:** 14 / 16 / 18 / 24 / 32 / 48

## Spacing & Layout

- **Spacing scale:** 4px base (4, 8, 12, 16, 24, 32, 48, 64)
- **Max content width:** 768px (single column, podcast-focused)
- **Breakpoints:** Mobile-first — 640px (sm), 768px (md), 1024px (lg)
- **Player:** Sticky bottom bar on mobile, inline on desktop

## Iconography

- Lucide React (consistent, lightweight, MIT licensed)
- Use sparingly — icons support text, never replace it

## Host Visual Identity

- **Aero:** Orange accent, angular avatar/icon, associated with critical/analytical UI elements
- **Nova:** Purple accent, rounded avatar/icon, associated with positive/growth UI elements
- Host ratings displayed side-by-side with their accent colors

## References

- Podcast apps (Spotify, Overcast) for player UX
- Product Hunt for product card layout
- Vercel/Linear for dark-mode aesthetic
