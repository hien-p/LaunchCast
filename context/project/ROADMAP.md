# Roadmap

> Generated by setup wizard from PRD. Aligned with hackathon phasing.

## Phase 1 — Core Pipeline + Vibe Setup (Day 1 Morning)

The foundation: end-to-end from scraping to audio output.

- F1: PH Scraping Pipeline (Firecrawl → product data)
- F2: Deep Product Website Analysis (Firecrawl → markdown per product)
- F3: Podcast Script Generation (Claude API → Aero/Nova conversation with ratings)
- F4: Voice Synthesis (ElevenLabs TTS → audio clips → FFmpeg stitching → MP3)
- F6: Vibe Setup screen (interest tags, host bios, voice preview)

**Deliverable:** End-to-end pipeline from Vibe Setup to playable audio.

## Phase 2 — Web Player + Generation Viewer (Day 1 Afternoon)

The visual layer: watch it build, then listen.

- F7: Live Episode Generation screen (SSE progress, real-time URLs, product cards)
- F5: Web Player with segmented progress bar and show notes
- F8: Product Deep Dive modal (PH blurb vs Firecrawl findings, per-host ratings)
- Mobile-responsive layout with sticky player

**Deliverable:** Full visual pipeline — watch the episode being built, then listen with show notes.

## Phase 3 — Live Mode + Trend Radar (Day 2 — Demo Polish)

The wow factor: interactive voice chat and weekly intelligence.

- F9: Live Interactive Mode (ElevenAgents, Firecrawl as Server Tool, visible search)
- F10: Trend Radar (weekly category trends, host picks, agreement rate)
- Share cards with OG images per segment

**Deliverable:** Full demo flow in 60 seconds.

## Phase 4 — Polish & Distribution (Post-Hackathon)

Growth and automation.

- Daily cron automation (6 AM generation)
- RSS feed for podcast apps (Apple Podcasts, Spotify)
- Episode archive and search
- Email/push notifications for new episodes
- Personalized episodes based on stored Vibe Setup preferences

## Backlog

- User accounts and authentication
- Sponsorship/monetization features
- Multi-language episode support
- Custom voice selection
- API for third-party integrations
