# Project Scope

> Generated by setup wizard from PRD. Updated as priorities shift.

## In Scope — v1 (Hackathon Demo)

### P0 — Must Have
- **F1: PH Scraping Pipeline** — Firecrawl scrapes top 5-10 daily PH launches
- **F2: Deep Product Website Analysis** — Firecrawl visits each product site, extracts features/pricing/audience
- **F3: Podcast Script Generation** — Claude generates Aero/Nova conversation with ratings and disagreements
- **F4: Voice Synthesis** — ElevenLabs TTS for two voices + FFmpeg audio stitching
- **F6: Vibe Setup** — Onboarding screen with topic selection, host bios, voice preview
- **F7: Live Episode Generation** — SSE-powered real-time progress (shows Firecrawl URLs being scraped)
- **F8: Product Deep Dive** — Modal with PH blurb vs Firecrawl findings side-by-side, per-host ratings

### P1 — Should Have (if time allows)
- **F5: Web Player & Episode Archive** — Basic player with show notes and product links
- **F9: Live Interactive Mode** — Voice chat with Aero & Nova via ElevenAgents
- **F10: Trend Radar** — Weekly category trends, host picks, agreement rate

## Out of Scope — v1 (possible later)

- User accounts and authentication
- Email/push notifications for new episodes
- RSS feed for podcast apps (Apple Podcasts, Spotify)
- Episode search and full-text search
- Personalized episodes based on individual user preferences (beyond localStorage)
- Daily cron automation (manual trigger in v1)
- SEO optimization and OG image generation
- Analytics dashboard
- Mobile native app
- Multi-language support

## Never In Scope

- Paid content or paywalls on episodes
- User-generated content (comments, reviews)
- Social features (following, profiles)
- Direct Product Hunt API integration (if it requires partnership/approval)
- Replacing or competing with Product Hunt itself

## Open Questions

- Podcast length: 10 minutes (top 5 products) or 20 minutes (top 10)?
- Music bed: License royalty-free intro/outro or generate with AI music?
- Product Hunt data source: Scrape site directly or use API/RSS if available?
- ElevenLabs voice selection: Which specific voice IDs for Aero and Nova?
